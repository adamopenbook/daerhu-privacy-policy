隱私權政策（最新版）

大耳狐故事屋（以下稱「本 App」）重視您的隱私。本政策說明本 App 如何處理資料與權限。

我們蒐集的資料

重要聲明：本 App 不收集任何個人識別資訊，包括但不限於：

  • 姓名、身分證字號、生日、Email 地址
  • 電話號碼、地址、位置資訊
  • 照片、影片或其他個人媒體檔案
  • 聯絡人資訊、通訊錄
  • 裝置識別碼或廣告識別碼

為提供語音合成功能，故事文字會送至第三方雲端服務進行轉換，但這些資料：

  • 不會用於使用者個人識別
  • 不會被永久儲存
  • 僅用於即時語音合成處理

第三方服務

本 App 使用以下第三方服務來提供功能：

  • 語音合成（TTS）：Microsoft Azure Cognitive Services
    • 用途：將故事文字轉換為語音
    • 資料處理：僅處理故事文字內容，不涉及個人資訊
    • 資料保留：不永久儲存，僅用於即時處理
  • AI 故事生成：OpenAI API
    • 用途：生成適合兒童的故事內容
    • 資料處理：僅處理故事生成請求，不涉及個人資訊
    • 資料保留：不永久儲存，僅用於即時處理

這些第三方服務的資料處理方式依其各自的服務條款與隱私權政策。我們選擇這些服務時已確認其符合兒童隱私保護標準。

權限使用目的

  • 網路（INTERNET）：向雲端 API 請求故事與語音。
  • 背景音訊：讓故事在螢幕關閉時持續播放。
  • 通知（Android 13+）：顯示播放控制訊息列（如已啟用）。

本 App 不會要求聯絡人、定位、相機、麥克風等與核心功能無關之權限。

兒童隱私保護

COPPA 合規聲明

本 App 完全符合美國《兒童線上隱私保護法》（COPPA）要求：

  • 不收集 13 歲以下兒童的個人資訊
  • 不要求兒童提供任何個人識別資訊
  • 不與第三方分享兒童資料
  • 提供家長控制功能

家長閘功能

為保護兒童安全，本 App 內建家長閘功能：

  • 在執行外部連結、分享或可能離開本 App 的操作時，會出現「家長確認」步驟
  • 需要長按按鈕並完成簡單數學驗證
  • 此設計用於降低兒童誤觸離開 App 的風險
  • 確保所有外部操作都經過家長同意

資料保存與安全

資料儲存政策

  • 本 App 不建立使用者帳戶
  • 不在本機或伺服器永久保存可識別個人之資料
  • 所有資料傳輸均使用 HTTPS 加密
  • 第三方服務的資料處理遵循其安全標準

資料刪除

  • 由於不收集個人資料，無需執行資料刪除程序
  • 故事內容和語音檔案僅暫存在裝置本地，可隨時清除
  • 若您透過 Email 與我們聯絡，郵件內容僅用於回覆需求，不會用於其他目的

法規遵循

台灣個資法（PDPA）

本 App 完全遵循台灣個人資料保護法：

  • 不收集個人資料，因此無需進行個資法告知程序
  • 若未來因功能需要蒐集個人資料，將依 PDPA 原則最小化並明確告知
  • 提供完整的查詢、更正或刪除途徑

GDPR 合規

本 App 亦符合歐盟一般資料保護規則（GDPR）要求：

  • 不收集個人資料，無需進行 GDPR 告知程序
  • 尊重用戶的資料保護權利
  • 提供透明的資料處理說明
